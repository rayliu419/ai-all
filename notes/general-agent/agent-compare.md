## agent比较
- coding agent 
  - claude code，cursor等。
  - 最成熟的agent。
  - 主要目标：读懂复杂代码库、跑测试、修 Bug、直接开 PR 合并代码。
  - 交互方式: 前台交互式，在终端守着它，或者丢一个让他限时去改代码。
- 企业工作流与自动化agent
  - n8n。
  - 用户用自然语言描述一个业务场景(“帮我监控竞品推特，发现重要更新就总结并同步到 Slack，同时给销售发一封邮件”)，Agent会自动理解意图、拆解步骤并动态调用各种API。
- 操作系统级别的agent。
  - hermes，通用的、可自我进化的任务型 Agent。
  - 工业级智能体框架。
  - 一个拥有长期记忆、能自己学习新技能、且可以无限分裂子智能体（Sub-agents）的分布式助理系统。
  - 交互式: 后台常驻式，结合Cron定时器，默默挂在后台、Telegram或服务器上24小时运行。
