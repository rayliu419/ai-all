方向导数与梯度 · 渐进式讲解

z = x² + y²  ·  从偏导到方向导数再到梯度
🎯 实例：碗壁上的行走

假设你现在站在碗 z = x² + y² 上的点 (1, 2)，此时的高度 z = 1² + 2² = 5。

该点的梯度为：∇f(1, 2) = (2·1, 2·2) = (2, 4)。 模长（最陡坡度）：|∇f| = √(2²+4²) = √20 ≈ 4.47。

如果你选择不同的方向 u⃗ 迈出一步：

你的选择（方向 u） 方向导数计算（Duf = ∇f · u） 直观物理意义
⬆ 沿最陡方向往碗口爬 u = (1/√5, 2/√5)
(2,4)·(1/√5,2/√5) = 10/√5 = √20		 你迎着梯度走，感受到了当前位置最陡的坡度（约 4.47），每步上升最多。
🔄 顺着碗壁绕圈（等高线） u = (−2/√5, 1/√5)
(2,4)·(−2/√5,1/√5) = 0		 你和梯度垂直，相当于在碗壁上水平绕圈。高度不随这一步发生任何改变。
⬇ 沿最陡方向往碗底滑 u = (−1/√5, −2/√5)
(2,4)·(−1/√5,−2/√5) = −√20		 你背离梯度走，迎来了最陡的下坡，直接冲向碗底，每步下降最多。
➡ 正 x 轴方向 u = (1, 0)
(2,4)·(1,0) = 2		 你斜着切上去，坡度变缓了。（偏导数 ∂z/∂x 其实就是沿 x 轴方向的方向导数。）
💡 核心关系：方向导数 Duf = ∇f · u = |∇f|·cos α。
当 α = 0°（与梯度同向）→ D = |∇f| = 4.47（最大上升）
当 α = 90°（与梯度垂直）→ D = 0（等高线方向，无升降）
当 α = 180°（与梯度反向）→ D = −|∇f| = −4.47（最大下降）
