一、矩阵乘法与列视角

核心骨架：矩阵乘法的"列视角"是神经网络前向传播的本质；"四个子空间"决定了数据在线性空间中的维数和解的存在性。

第 2 章：矩阵乘法 & 逆矩阵

矩阵乘法不止一种理解方式，但最重要的视角是列视角：

  • 行视角：$C = AB$ 中 $C$ 的每一行是 $B$ 的行向量的线性组合（系数来自 $A$ 的对应行）
  • 列视角：$C$ 的每一列是 $A$ 的列向量的线性组合（系数来自 $B$ 的对应列）

列视角是机器学习中理解神经网络层变换的关键 — 一个权重矩阵 $W$ 乘以输入 $x$，本质上是将 $x$ 作为系数对 $W$ 的列向量做线性组合，输出落在 $W$ 的列空间里。

第 3 章：向量空间、零空间、四个子空间

对于一个 $m \times n$ 矩阵 $A$，它定义了四个基本的向量子空间，这四个子空间完整刻画了 $A$ 的线性变换行为。在 ML 中，它们决定了：

  • 数据的有效维度（秩）
  • 方程 $Ax = b$ 解的存在性和唯一性
  • 降维和压缩的理论极限

二、四个基本子空间

对于一个 $m \times n$ 的矩阵 $A$（$m$ 行，$n$ 列），四个子空间分别是：

子空间 定义 维度 所在空间
列空间 Column Space $C(A) = \{ Ax \mid x \in \mathbb{R}^n \}$ $r$（秩） $\mathbb{R}^m$
零空间 Null Space $N(A) = \{ x \mid Ax = 0 \}$ $n - r$ $\mathbb{R}^n$
行空间 Row Space $R(A) = C(A^\mathsf{T})$ $r$ $\mathbb{R}^n$
左零空间 Left Null Space $N(A^\mathsf{T}) = \{ y \mid A^\mathsf{T} y = 0 \}$ $m - r$ $\mathbb{R}^m$
核心关系：
  • 行空间与零空间在 $\mathbb{R}^n$ 中正交且互补：$\dim R(A) + \dim N(A) = n$
  • 列空间与左零空间在 $\mathbb{R}^m$ 中正交且互补：$\dim C(A) + \dim N(A^\mathsf{T}) = m$
  • $Ax = b$ 有解 $\iff$ $b \in C(A)$

三、列空间（Column Space）$C(A)$

定义：$A$ 的所有列向量的线性组合构成的集合。

维度：等于 $A$ 的秩 $r$，即线性无关列的最大数量。

几何意义

列空间是矩阵 $A$ 作为线性变换的像空间（range / image）。$A$ 将 $\mathbb{R}^n$ 中的向量映射到 $\mathbb{R}^m$ 中，但输出只填满一个 $r$ 维的子空间，而不是整个 $\mathbb{R}^m$（除非 $r = m$，即满秩）。

ML 应用

  • 神经网络的层变换：权重矩阵 $W^{(l)}$ 的列空间决定了该层可以表达的输出的范围。如果列空间的秩不足，网络表达能力受限。
  • 线性回归：设计矩阵 $X$ 的列空间包含了所有可能的预测值 $\hat{y} = X\hat{\beta}$。最小二乘解 $\hat{\beta}$ 将 $y$ 投影到 $C(X)$ 上。
  • PCA 降维：保留最大方差的 $k$ 个方向等价于在列空间中找一个 $k$ 维子空间来近似原始数据。

四、零空间（Null Space）$N(A)$

定义：所有满足 $Ax = 0$ 的向量 $x$ 的集合。

维度：$n - r$，即$n$列数减去秩。称为零度（nullity）。

几何意义

零空间是被 $A$ "压扁"到零的那些输入方向。维度越大，表示越多的输入信息被矩阵变换丢弃了。如果 $A$ 是满列秩（$r = n$），则零空间只有零向量，映射是单射（injective）。

ML 应用

  • 欠定系统：当样本数 $m$ 小于特征数 $n$ 时，$Ax = b$ 有无限多解，零空间的非零向量对应了解的"自由分量"。
  • 正则化：L2 正则化（Ridge）通过惩罚 $\|x\|_2$ 在零空间中挑出一个特定解（最小范数解），消除歧义性。
  • 对抗样本：神经网络权重矩阵的零空间方向可以产生人类无感知但模型完全误判的对抗扰动。

五、行空间（Row Space）$R(A)$

定义：$A$ 的所有行向量的线性组合构成的集合。等价于 $C(A^\mathsf{T})$。

维度：$r$（与列空间相同）。

几何意义

行空间和零空间都是 $\mathbb{R}^n$ 的子空间，且互相正交。行空间包含了矩阵 $A$ 在输入空间中的"有效方向" — 沿着这些方向的输入会被 $A$ "看到"并映射到输出空间；沿着零空间方向的输入则被忽略。

关键关系：$A$ 的行空间和零空间是 $\mathbb{R}^n$ 中的正交补（orthogonal complements）： $$ \mathbb{R}^n = R(A) \oplus N(A) $$ 任何一个 $\mathbb{R}^n$ 中的向量都可以唯一分解为一个行空间分量加一个零空间分量。

ML 应用

  • 特征选择：行空间告诉我们在输入特征中哪些方向是有效的。如果某行与其他行线性相关，对应的特征可能是冗余的。
  • SVD 解释：$A = U\Sigma V^\mathsf{T}$ 中，$V^\mathsf{T}$ 的行（$V$ 的列）张成了 $A$ 的行空间。

六、左零空间（Left Null Space）$N(A^\mathsf{T})$

定义：$A^\mathsf{T}y = 0$（等价于 $y^\mathsf{T}A = 0$，即左乘 $A$ 得零行向量）的所有 $y$ 的集合。

维度：$m - r$。

几何意义

列空间和左零空间是 $\mathbb{R}^m$ 中的正交补。左零空间包含了所有输出空间中的方向，这些方向完全不可达 — 没有任何输入 $x$ 可以使 $Ax$ 指向左零空间的非零方向。

ML 应用

  • 残差分析：在线性回归中，残差 $e = y - \hat{y}$ 位于左零空间（因为 $\hat{y}$ 在 $C(X)$ 中且 $e \perp C(X)$）。
  • 约束条件验证：左零空间给出了线性系统 $Ax = b$ 存在的相容性条件：$b$ 必须与 $N(A^\mathsf{T})$ 正交才有解。

七、总结与关系图

一句话总结：四个子空间完整刻画了一个矩阵作为线性变换的全部行为 — 哪些输入方向有效（行空间）、哪些被丢弃到零（零空间）、能到达输出的哪些方向（列空间）、哪些输出方向永远去不了（左零空间）。

直观记忆

四个子空间的关系（$m \times n$ 矩阵 $A$，秩 $r$）：

         ℝⁿ               ℝᵐ
    ┌──────────┐      ┌──────────┐
    │ Row Space│  A   │Col Space │
    │ dim = r  │─────▶│ dim = r  │
    │          │      │          │
    │  ⟂       │      │  ⟂       │
    │Null Space│      │Left Null │
    │dim = n-r │      │dim = m-r │
    └──────────┘      └──────────┘

$\dim R(A) = \dim C(A) = r$（秩）
$\dim R(A) + \dim N(A) = n$
$\dim C(A) + \dim N(A^\mathsf{T}) = m$
