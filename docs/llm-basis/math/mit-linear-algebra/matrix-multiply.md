## 逆矩阵
- 对于方阵，如果存在逆矩阵，左逆矩阵必然等于右逆矩阵。
- 奇异矩阵，无逆矩阵。
- 什么时候没有逆矩阵？也有多个解释。
  - 对应的行列式为0。
  - 可以找到一个**非零向量x**, 使Ax = 0。
    - 反证法证明，假设逆矩阵存在，都左乘A的逆矩阵 => x = 0 
    - 矛盾。
- 求矩阵的逆矩阵其实与计算方程组的解是一样的。
  - A * B = I 
  - 以列视角来看，B的每一列相当于x，I对应的列则是某个位置为1的列。
  - 所以求B的每一列，相当于求多个x。
  - 系数是一样的，x和b不一样。
  - Gauss-Jordan方法，一次解决多个方程组。
  - 把I合并到增广矩阵中，一起使用Gauss-Jordan方法， 在左边变为I时，右边就成是A逆。
    - why? 
    - 1. 对一个矩阵进行初等行变换，等价于在它的左边乘以一个对应的初等矩阵。
    - 2. 在 Gauss-Jordan 消元法中，我们对增广矩阵$(A \mid I)$进行了一系列连续的行变换。假设我们依次做了第 1, 2, ..., $k$步变换，我们可以把这所有步骤合并成一个总变换矩阵，记为$P$
    - 3. 因为我们是对整个增广矩阵 $(A \mid I)$ 的每一行进行同步变换，所以这相当于用矩阵 $P$ 去左乘整个增广矩阵：$P (A \mid I) = (PA \mid PI)$
    - 4. 左边的部分变为了$I$，即：$PA = I$根据可逆矩阵的定义，如果$PA = I$，那么$P$就一定是$A$的逆矩阵！$P = A^{-1}$。此时，我们再看右边的部分。右边原本是$PI$，既然$P = A^{-1}$，那么右边自然就变成了：$PI = P = A^{-1}$所以，当左边被化为$I$时，整个增广矩阵就变成了：$(PA \mid PI) \implies (I \mid A^{-1})$这就是为什么右边会自动“显现”出$A^{-1}$的原因。
