## 向量空间
- 向量有什么操作？
  - 向量相加
  - 数乘
- 向量定义
  - 相加向量，数乘和线性组合必须仍在这个空间。
- 子空间定义
  - 标准的R^n显然都是向量空间。
  - 但是能不能从里面抽取一些子集，也是向量子空间。
- 直观上找出一些子空间
  - 对数乘封闭，意味着它的是一条直线。
  - 但是得经过原点。
  - 只有部分子集才是子空间。
- R^2的子空间有哪些？
  - R^2自己。
  - 经过原点的直线。
  - 只包含零向量的空间。
- R^3的子空间
  - R^3自己。
  - **经过原点的平面。**
  - 经过原点的直线。
  - 只包含零向量的空间。
- 从矩阵A构造子空间。
  - 假设有一个3 * 2的矩阵A，取它们的列作为两个列向量。
  - 这两个列向量需要对加法，数乘和线性组合封闭。
  - **都满足的运算结果构成了这两个列向量的子空间，称为列空间，记为C(A)。**
  - 结果是什么？
    - 是一个通过原点的平面。
