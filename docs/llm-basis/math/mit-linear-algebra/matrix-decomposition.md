## A的LU分解
- A = L * U 
  - L = lower
  - U = upper
- 在初学线性代数时，高斯消元法已经能很好地解方程组了，为什么还要大费周折地把一个矩阵$A$拆成一个下三角矩阵$L$和一个上三角矩阵$U$（即$A = LU$）呢？
- 简而言之：高斯消元法是“手动挡”，而$LU$分解是“自动挡”和“工业级方案”。 它的核心意义在于高效率、高复用性、以及更低的计算成本。
- 在实际工程（如桥梁受力分析、电路模拟、图像处理）中，我们经常需要求解一系列方程组:$Ax = b_1, \quad Ax = b_2, \quad Ax = b_3, \dots$
- 这里的系数矩阵 $A$ 是完全一样的（代表系统结构没变），但右边的常数项 $b$ 在不断变化（代表外力、输入信号或时间在变）。
- 如果用$LU$分解：你只需要对$A$做一次$LU$分解（成本为 $O(n^3)$，后面就一劳永逸了）。
- 对于每一个新的$b$，方程$Ax = b$变成了$LUx = b$。我们可以拆成两步极快的“回代”来解：
  - 令 $Ux = y$，先解下三角方程：$$Ly = b \quad (\text{用前向回代法法，Forward Substitution})$$
  - 求出 $y$ 后，再解上三角方程：$$Ux = y \quad (\text{用后向回代法，Back Substitution})$$
  - 因为 $L$ 和 $U$ 是三角矩阵，解这两个方程不需要任何消元，只需要一步步带入即可。解一次的计算量仅为 $O(n^2)$。
